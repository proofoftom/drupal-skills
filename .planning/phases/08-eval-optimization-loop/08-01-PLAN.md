2. Target directory: `/tmp/d10-${NAME}`
3. `unset CLAUDECODE` at top (per MEMORY.md rule #8 -- prevents nested session blocking)

10. Verify bootstrap: `ddev drush status --field=bootstrap | grep -q "Successful"`
11. On success, echo the target directory path
12. On any failure, print FATAL message to stderr and exit 1
13. Make executable: chmod +x
