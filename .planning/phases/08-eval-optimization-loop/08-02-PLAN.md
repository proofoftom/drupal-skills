**Step 3: eval-browser validation (INFRA-04)**
Spawn eval-browser agent:
- Task: "Verify the Drupal site at ddev project d10-smoke-test is working. Log in via drush uli --uri=https://d10-smoke-test.ddev.site, take a snapshot of the front page, and report the page title and any visible content."
- After agent completes: confirm it successfully reported page content

**Step 4: eval-grader validation (INFRA-02)**

- ddev project: `d10-smoke-test`
- After agent completes: verify output matches grading.json schema (has expectations array with text/passed/evidence, has summary with passed/failed/total/pass_rate)

**Step 5: Teardown**

Verify the environment is cleaned up.
