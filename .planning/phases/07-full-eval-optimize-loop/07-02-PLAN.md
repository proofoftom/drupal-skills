5. Write transcript to workspace transcript.md

Execute via:

**Step 1: Grade 6 runs**

For each of the 6 runs, read transcript.md and output files, then grade against expectations from the skill's evals.json. Write grading.json in the standard format:

```json
{
  "expectations": [
    { "text": "assertion text from evals.json", "passed": true, "evidence": "Found in file X: [relevant code snippet]" },
    { "text": "...", "passed": false, "evidence": "Searched all output files, pattern not found" }
  ],
  "summary": { "passed": N, "failed": N, "total": N, "pass_rate": 0.XX }
}
```

Grading rules:
- Read ALL output files for each run (not just one file)
- For each assertion, provide specific evidence (file path + code snippet or command output)
- `passed: null` only if the assertion was NOT testable (infrastructure issue)
- Be generous: if the intent is met even with minor variations, pass it
