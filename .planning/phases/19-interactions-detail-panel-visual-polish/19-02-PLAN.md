In edit mode: `<input ref="editInput" v-model="editValue" class="gapm-task-card__title-input" @keydown.enter="saveEdit" @keydown.escape="cancelEdit" @blur="saveEdit" />`.
    Use a 200ms click delay pattern to distinguish single-click (open panel) from double-click (edit). On single click, set a timer. If dblclick fires within 200ms, clear timer and activate editing. If timer completes, emit `open-panel`.
