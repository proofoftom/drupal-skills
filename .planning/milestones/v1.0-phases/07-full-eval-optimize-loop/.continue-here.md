## Key Findings This Session

3. **Sonnet knows standard Drupal patterns cold**: Routing, forms, blocks all show 0% delta even with differentiating assertions. The assertions from plan 07-06 (no \Drupal::service, interface types, 4-param create, blockForm/blockSubmit, schema types) are NOT differentiating — Sonnet gets them all right without the skill.

4. **Clean teardown order**: Must `drush pm:uninstall` BEFORE `rm -rf` module files, or Drupal registry gets corrupted. For eval reuse, tear down entire ddev env instead.

5. **E2E port issue**: e2e-assert.sh uses default HTTPS URL but ddev assigns non-standard ports. Use curl for E2E verification.
