6. Entity handlers include form, list_builder, access handler, and route_provider declarations
7. Module enables successfully: ddev drush en knowledge_resource returns exit code 0

**Testing expectations** (from skills/drupal-testing/evals/evals.json — 8 assertions):
1. Test class extends KernelTestBase (NOT BrowserTestBase or WebDriverTestBase)
2. Test class has protected static $modules array listing required modules
3. $modules includes 'node' and 'social_ai_indexing'
4. setUp() calls parent::setUp()
5. setUp() calls $this->installEntitySchema('node') or similar entity schema installation
6. Test has @group annotation
7. Test uses $this->container->get('social_ai_indexing.related_content') or \Drupal::service() to load the service
8. Test uses proper assertion methods ($this->assertNotNull, $this->assertEquals, etc.)
