  <action>
Rewrite `DashboardController::content()` to:

1. Query all projects via `$this->entityTypeManager->getStorage('project')` with `accessCheck(TRUE)` and `sort('created', 'DESC')`.
2. For each project, load its tasks via `$this->entityTypeManager->getStorage('task')->loadByProperties(['project' => $project->id()])`.
3. Count tasks per status: `['todo' => 0, 'in_progress' => 0, 'review' => 0, 'done' => 0]`. Use `$task->getStatus()` (falls back to 'todo' if null).
4. Calculate progress: `round(($summary['done'] / $total) * 100)` (0 if no tasks).
5. Build `$project_cards` array with keys: `title`, `status`, `url` (via `$project->toUrl()->toString()`), `board_url` (via `Url::fromRoute('group_ai_pm.kanban_board', ['project' => $project->id()])->toString()`), `summary` (the counts array), `total`, `progress`.
6. Build quick actions array with `new_project` entry: `['title' => $this->t('New Project'), 'url' => Url::fromRoute('entity.project.add_form')->toString()]`.
7. Build `$recent_projects` as `array_slice($project_cards, 0, 5)` for quick-access links.
8. Return render array using `'#theme' => 'group_ai_pm_dashboard'` with `#project_cards`, `#quick_actions`, `#recent_projects`.
9. Attach library: `'#attached' => ['library' => ['group_ai_pm/dashboard']]`.
10. Add cache tags: `['project_list', 'group_ai_pm_task_list']` plus each project's own cache tags via `$project->getCacheTags()`. Add cache context `'user.permissions'`.

Add `use Drupal\Core\Url;` import at top.
