**Step 1: Create useKanban.js composable** (`modules/group_ai_pm/js/src/composables/useKanban.js`):

Manages reactive board state and API operations. Uses Vue's `ref` and `computed` (imported from 'vue').

```javascript
import { ref, computed } from 'vue';
import { apiMutate } from '../api/drupal.js';
```

Export function `useKanban(initialTasks, apiBaseUrl, csrfTokenUrl)`:
- `tasksByStatus` = `ref({...initialTasks})` (deep clone initial data)
- `loading` = `ref(false)`
- `error` = `ref(null)`

Methods:
- `getColumnTasks(statusId)` -> returns computed or getter for `tasksByStatus.value[statusId] || []`
- `moveTask(taskId, fromStatus, toStatus)`:
  1. Optimistic update: remove task from fromStatus array, update its status field, add to toStatus array
  2. Call `apiMutate('PATCH', apiBaseUrl + '/task/' + taskId + '/status', { status: toStatus }, csrfTokenUrl)`
  3. On error: roll back (move task back to fromStatus, restore original status). Set `error.value` with message.
- `createTask(title, status)`:
  1. Set loading
  2. Extract projectId from the API base URL pattern or accept as parameter
  3. Call `apiMutate('POST', apiBaseUrl + '/project/' + projectId + '/task', { title, status }, csrfTokenUrl)`
  4. On success: add returned task to `tasksByStatus.value[status]`
  5. On error: set `error.value`
- `handleDragEnd(event)`: Extract the moved task's new column (by reading the column component's status prop or from event data), call `moveTask` with old and new status.
- `clearError()`: Sets error.value to null.

Return: `{ tasksByStatus, loading, error, getColumnTasks, moveTask, createTask, handleDragEnd, clearError }`

IMPORTANT: The composable must accept `projectId` as a parameter (needed for createTask POST URL).

**Step 3: Create KanbanColumn.vue:**

Uses `VueDraggable` from vue-draggable-plus for drag-and-drop between columns.

```vue
<script setup>
import { ref, computed } from 'vue';
import { VueDraggable } from 'vue-draggable-plus';
import TaskCard from './TaskCard.vue';
import QuickCreateForm from './QuickCreateForm.vue';

const props = defineProps({
  column: { type: Object, required: true },
  tasks: { type: Array, required: true },
  allColumns: { type: Array, required: true },
  permissions: { type: Object, required: true },
  priorityLabels: { type: Object, required: true },
  apiBaseUrl: { type: String, required: true },
  csrfTokenUrl: { type: String, required: true },
});

const emit = defineEmits(['task-moved', 'task-created']);
const showQuickCreate = ref(false);
const localTasks = computed({
  get: () => props.tasks,
  set: () => {},
});

function onDragEnd(event) {
  // VueDraggable fires 'end' with from/to info
  // The task that was moved
  if (event.from !== event.to) {
    const taskId = event.item?.dataset?.taskId;
    const fromStatus = event.from?.dataset?.status;
    const toStatus = event.to?.dataset?.status;
    if (taskId && fromStatus && toStatus) {
      emit('task-moved', parseInt(taskId, 10), fromStatus, toStatus);
    }
  }
}

function onTaskCreated(title) {
  emit('task-created', title, props.column.id);
  showQuickCreate.value = false;
}

const taskCount = computed(() => props.tasks.length);
</script>

<template>
  <div class="gapm-kanban__column" :class="'gapm-kanban__column--' + column.id">
    <div class="gapm-kanban__column-header">
      <span class="gapm-kanban__column-title">{{ column.label }}</span>
      <span class="gapm-kanban__column-count">{{ taskCount }}</span>
      <button
        v-if="permissions.createTask"
        class="gapm-kanban__quick-create-btn"
        @click="showQuickCreate = !showQuickCreate"
        :aria-label="'Add task to ' + column.label"
        :title="'Add task to ' + column.label"
      >+</button>
    </div>
    <QuickCreateForm
      v-if="showQuickCreate"
      :status="column.id"
      @created="onTaskCreated"
      @cancel="showQuickCreate = false"
    />
    <VueDraggable
      v-model="localTasks"
      group="tasks"
      :animation="200"
      item-key="id"
      class="gapm-kanban__column-body"
      :data-status="column.id"
      @end="onDragEnd"
    >
      <template #item="{ element }">
        <TaskCard
          :task="element"
          :all-columns="allColumns"
          :priority-labels="priorityLabels"
          @status-change="(newStatus) => emit('task-moved', element.id, column.id, newStatus)"
        />
      </template>
    </VueDraggable>
    <div v-if="tasks.length === 0 && !showQuickCreate" class="gapm-kanban__empty">
      No tasks
    </div>
  </div>
</template>
```

IMPORTANT: The `data-status` attribute on the VueDraggable container and `data-task-id` on TaskCard root are needed for the drag-end handler to determine source/target columns and which task moved.

**Step 5: Create QuickCreateForm.vue:**

Inline title input for quick task creation (BOARD-09).

```vue
<script setup>
import { ref, nextTick, onMounted } from 'vue';

const props = defineProps({
  status: { type: String, required: true },
});

const emit = defineEmits(['created', 'cancel']);
const title = ref('');
const inputRef = ref(null);

onMounted(() => {
  nextTick(() => {
    if (inputRef.value) {
      inputRef.value.focus();
    }
  });
});

function submit() {
  const trimmed = title.value.trim();
  if (trimmed) {
    emit('created', trimmed);
    title.value = '';
  }
}

function cancel() {
  title.value = '';
  emit('cancel');
}
```
