    **drupal.js -- add deleteKanban():**
    Add `export async function deleteKanban(url)` that fetches CSRF token via existing `getCsrfToken()`, sends DELETE request with `X-CSRF-Token` header, checks `response.ok`, throws on error. Returns nothing (204 has no body). Follow the exact same pattern as `patchKanban()` but with `method: 'DELETE'`, no body, and no `response.json()` on success.

    **useToast.js:**
    Module-level `ref([])` for toast queue (shared singleton across all importers). Export `useToast()` returning `{ toasts, addToast, removeToast }`. `addToast(message, type = 'info', duration = 4000)` pushes `{ id: ++counter, message, type }` and sets `setTimeout` for auto-removal. Types: 'success' (3s), 'error' (5s), 'info' (4s). Store timeout IDs to clear on manual dismiss. Do NOT use `onUnmounted` cleanup since this is a singleton composable -- instead, track timeouts per toast ID and clear on removeToast.

    **useFilters.js:**
    Export `useFilters()` returning `{ filters, clearFilter, clearAll, hasActiveFilters }`.
    `filters` is `ref({ assignee: null, priority: null })`.
    On `onMounted()`, read `window.location.search` via `URLSearchParams` and set initial values.
    `watch(filters, ...)` with `{ deep: true }` updates URL via `history.replaceState()`. Build new URL from `window.location.pathname` + params. Delete params with null values.
    `hasActiveFilters` is a `computed` returning true if any filter is non-null.
    `clearFilter(key)` sets `filters.value[key] = null`.
    `clearAll()` resets all to null.

    **useContextMenu.js:**
    Export `useContextMenu()` returning `{ visible, position, task, show, hide }`.
    `visible` is `ref(false)`, `position` is `ref({ x: 0, y: 0 })`, `task` is `ref(null)`.
    `show(event, taskData)`: sets task, calculates position from `event.clientX/clientY`, sets visible true. Use `nextTick()` to check menu bounds against `window.innerWidth/innerHeight` and adjust if overflow. Close on document click listener (added in `show`, removed in `hide`). Close on Escape key.
    `hide()`: sets visible false, removes listeners.

    **useDisplayOptions.js:**
    Export `useDisplayOptions(projectId)` returning `{ options, toggleOption }`.
    `options` is `ref({ showPriority: true, showAssignee: true, showDueDate: true, compactMode: false })`.
    On creation, read from `localStorage` key `gapm-display-${projectId}`. If exists, `JSON.parse` and merge with defaults (handles missing keys gracefully).
    `watch(options, ...)` with `{ deep: true }` writes to localStorage.
    `toggleOption(key)` flips `options.value[key]`.
  </action>
